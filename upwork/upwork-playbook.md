# Upwork Playbook & Tracker

Goal: land paid gigs fast to fund product work. This is the money engine for the next 1-3 months. Optimize for win rate, not volume.

## Positioning

Do not present as a generalist. A profile listing Flutter, PHP, Next.js and "everything" reads as jack-of-all-trades and competes on price. Clients search for specialists and pay them more. Run one lead profile plus specialized profiles.

- Lead profile: Python/FastAPI + Next.js full-stack SaaS. Highest-rate core. Back it with real client work (custom e-commerce platform, accounts receivable system).
- Specialized profile: Flutter mobile development.
- Specialized profile (high priority): Crypto / Web3 / Solana bots and dashboards. Pays well, and the work doubles as reusable code for the trading tool.
- PHP / Laravel: not a headline skill. Bid only on well-budgeted, good-fit PHP gigs when they appear.

## Profile setup checklist

- [ ] Pick lead specialization and write a niche headline (not "full-stack developer")
- [ ] Add portfolio pieces with outcomes: Servaspot, e-commerce platform, accounts receivable system
- [ ] Create Flutter specialized profile
- [ ] Create Web3/Solana specialized profile
- [ ] Set job alerts per niche
- [ ] Set a rate per profile

## Which jobs to bid on

Bid:

- Posted within the last hour (early proposals convert far better)
- Clear scope and deliverables
- Payment method verified
- Reasonable budget
- Fits one of the specialized profiles

Skip (connects die here):

- 50+ proposals already
- Vague or one-line specs
- Payment method not verified
- Client with no hire history and a thin post

## Proposal rules

- Apply within the first hour of a posting.
- Tailor the first two lines. That is all the client sees in the preview, so lead with a relevant result or portfolio piece, not a template.
- Keep it short and specific to the job. No generic copy-paste.
- Reference the matching portfolio piece for that niche.

## Connects

- 100 connects costs ~$16.5. The cost is noise. Win rate is the constraint.
- 100 connects is roughly 6-15 proposals depending on job cost.
- Do not spray. Spend only on tight-fit jobs.
- Boosted proposals: only on high-value, high-fit jobs, and sparingly.
