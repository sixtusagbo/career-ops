The brief was a 30-minute build: take a job title, return three interview questions via an AI API, plus a 4–7 minute Loom walking through the build and answering four reflection questions. Built with Next.js 16, TypeScript, Tailwind v4, OpenAI Responses API with Zod for structured outputs, deployed on Vercel. Used gpt-5-nano. Added an "intent" field to each question beyond the spec, to make the output useful to the interviewer rather than just the candidate.
